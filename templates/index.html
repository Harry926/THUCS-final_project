Twitter Search:

How does Username work

使用Twitter Api 查詢公開的推特數據來確認其他人與你的互動多寡


它會獲取你最近7天內的推文(至多50篇)推文以及其中的喜歡、留言和轉推來計算你與每個人的互動次數並打分數。


Interaction Score
喜歡 1
轉推 1
留言 2

查詢的帳號會處在最中心，而其他所有人會依照得分被分類成圈，得分前10位在第一圈，第11位到30位在第二圈，後三十位在第三圈。

點擊情緒按鈕將顯示最近推文的情緒最多的為何種顏色


接著獲取這些人7天內發的所有推文，並透過訓練好的情緒分類器
進行分類。並按照情緒的多寡改變頭像外框顏色。


此外點擊頭像會顯示詳細資料可以看到7天情緒變化的折線圖以及情緒統計的圓餅圖，也可以看到其他人在7天內對你的喜歡、轉推和留言的數量。


How does Keyword work

使用Twitter Api 查詢公開的推特數據


輸入關鍵字後會去找此關鍵字的近1000篇推文的原始推文，原始推文表示此推文不是轉推、留言或引用轉推，而是使用者所發的推文。 接著再對這1000篇推文進行情緒分析
下一頁將顯示關鍵字查詢到的推文內文，和分析出來的情緒的折線圖以及統計的圓餅圖

How does Reply work

使用Twitter Api 查詢公開的推特數據


輸入你想要分析的推文網址，接著將會抓取這篇推文當中的所有回覆
以及引用轉推並對這些回覆進行情緒分析
下一頁將顯示回覆內文，和情緒的折線圖以及統計的圓餅圖
